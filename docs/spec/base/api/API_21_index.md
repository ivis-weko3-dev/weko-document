## ツリー編集画面表示
### 目的・用途

インデックスツリーを表示するための画面を表示する

### 利用方法

利用可能なロールでログインしたセッション情報を用いて実行する

* インデックスを指定しない場合

  ```shell
  curl -X GET <WEKO3のURL>/admin/indexedit/ -h "Cookie: session=<セッション情報>"
  ```

* インデックスを指定する場合

  ```shell
  curl -X GET <WEKO3のURL>/admin/indexedit/<index_id> -h "Cookie: session=<セッション情報>"
  ```

### 利用可能なロール

|ロール|システム<br>管理者|リポジトリ<br>管理者|コミュニティ<br>管理者|登録ユーザー|一般ユーザー|ゲスト<br>(未ログイン)|
|---|---|---|---|---|---|--|
|利用可否|○|○|○|×|×|×|

### 機能内容

- ツリー編集の画面を表示する

### 関連モジュール

- weko_index_tree

### 処理概要

1. `WEKO_ACCOUNTS_SHIB_BIND_GAKUNIN_MAP_GROUPS`がTrueである場合、キャッシュ
DBよりmAPに登録されているグループ情報を取得し、WEKOシステム内にロールが存在しなければ追加し、キャッシュDBに存在しなければ削除する。  
この際、インデックスへの閲覧権限・投稿権限も設定値に応じて更新する。
1. `WEKO_INDEX_TREE_INDEX_ADMIN_TEMPLATE`に定義されたページを表示する。

## 画像ファイルアップロード
### 目的・用途

画像ファイルをサーバーにアップロードする

### 利用方法

利用可能なロールでログインしたセッション情報を用いて実行する

```shell
curl -X POST <WEKO3のURL>/admin/indexedit/upload -h "Cookie: session=<セッション情報>" -F "file=@<アップロードするファイルのパス>"
```

### 利用可能なロール

|ロール|システム<br>管理者|リポジトリ<br>管理者|コミュニティ<br>管理者|登録ユーザー|一般ユーザー|ゲスト<br>(未ログイン)|
|---|---|---|---|---|---|--|
|利用可否|○|○|○|×|×|×|

### 機能内容

- インデックスのサムネイル画像として用いるファイルをサーバーにアップロードする

### 関連モジュール

- weko_index_tree

### 処理概要

1. リクエストボディにファイルが含まれていない場合、またはファイルが選択されていない場合はその旨をメッセージとして出力したうえでエラー終了する。
1. 設定された場所へアップロードされたファイルを保存する。
1. DBに登録する用のパスを生成し、呼び出し元に返却する。
